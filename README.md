# 运动记录可编辑面板

一个仿“运动记录”风格的单页应用，支持你自由修改数据并实时刷新界面。

## 使用方式

1. 用浏览器打开 `index.html`。
2. 右侧“数据编辑器”直接修改 JSON。
3. 点击“应用数据”后，左侧面板立即更新。
4. 数据会自动保存在浏览器 `localStorage`。

## 可改字段

- `year`: 年份
- `totals`: 顶部三张卡片数据
- `yearSummary`: 年度汇总指标
- `monthlyDistance`: 12 个月柱状图里程数组
- `latestRun`: 最近一次跑步信息

## 提示

- JSON 必须是合法格式，否则会显示错误。
- 点击“恢复默认”可快速回到初始数据。
